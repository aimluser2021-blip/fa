# HMF Chemical Enterprises Starter Project

This is a lightweight starter app for your chemical business. It includes:

- Dashboard with sales and stock summary
- Billing form with invoice preview
- Inventory page with low-stock status
- Customer records
- Reports page with payment and product summary
- Your HMF logo already connected

## Run locally

1. Open PowerShell in this folder.
2. Run:

```powershell
python run_app.py
```

3. Open this address in your browser:

```text
http://127.0.0.1:8000
```

## Notes

- Data is stored in browser local storage for demo purposes.
- `Reset Demo Data` restores the sample records.
- `Export JSON` downloads your current demo/business data.

## Project files

- `index.html` - app layout
- `styles.css` - app styling
- `app.js` - business logic and demo data
- `run_app.py` - local server
- `app-logo.svg` - editable logo
